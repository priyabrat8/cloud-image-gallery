# 📸 Cloud Image Gallery (Local Storage)

A simple image hosting web application built with **Node.js**, **Express.js**, and **Multer**. Users can upload images, which are stored locally on the server and displayed in a responsive gallery.

---

## ✨ Features

- 📤 Upload images through a simple web interface
- 🗂 Store uploaded images locally using Multer
- 🖼 Display uploaded images in a gallery
- ⚡ Fast and lightweight Express backend
- 📱 Responsive user interface

---

## 🛠 Tech Stack

### Frontend
- Handlebars
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### File Upload
- Multer

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/priyabrat8/image-market.git
```

Move into the project directory

```bash
cd image-market
```

Install dependencies

```bash
npm install
```

Start the application

```bash
npm start
```

Visit

```
http://localhost:3000
```
